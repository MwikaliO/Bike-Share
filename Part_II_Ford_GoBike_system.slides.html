Features Determining Bike Ride Durations¶

Investigation Overview¶

In this investigation, I wanted to look at the features that could be used to predict bike ride durations. The main focus was on the Birth year of the rider,User type, Gender,hours and days of the week,Bike share.

Dataset Overview¶

  • This data set consist of 183 000 rides made in a bike-sharing system covering the greater San Francisco Bay area. The dataset include the following features rider duration, Stations, time, gender of the rider and also user type.
  • The dataset had some missing values in some row were removed.
  • The start time and end time column had some data that were extracted lead to addition of number of columns
